详细描述

Wrapper around RMMapContents for the iPhone.

It implements event handling; but that's about it. All the interesting map logic is done by RMMapContents. There is exactly one RMMapView instance for each RMMapContents instance.

A -forwardInvocation method exists for RMMap, and forwards all unhandled messages to the RMMapContents instance.

Bug:
No accessors for enableDragging, enableZoom, deceleration, decelerationFactor. Changing enableDragging does not change multitouchEnabled for the view.
